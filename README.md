# Cryptography
# C++ Cryptography Algorithms

A comprehensive collection of cryptographic algorithms implemented in C++, designed for students, developers, cybersecurity enthusiasts, and researchers who want to explore the principles of cryptography through practical implementations.

This repository contains both classical and modern cryptographic algorithms, along with hashing functions, key exchange mechanisms, and digital signature schemes. Each implementation focuses on readability, efficiency, and educational value while following modern C++ programming practices.

## Features

### Classical Cryptography

* Caesar Cipher
* ROT13 Cipher
* Vigenère Cipher
* Playfair Cipher
* Rail Fence Cipher
* Affine Cipher
* Hill Cipher
* Columnar Transposition Cipher

### Modern Cryptography

* AES (Advanced Encryption Standard)
* DES (Data Encryption Standard)
* Triple DES (3DES)
* RSA Encryption
* ElGamal Encryption
* Diffie-Hellman Key Exchange

### Hashing Algorithms

* MD5
* SHA-1
* SHA-256
* SHA-512

### Authentication & Digital Signatures

* HMAC
* RSA Digital Signatures
* Message Integrity Verification

### Utility Tools

* Key Generation
* Random Number Generation
* File Encryption & Decryption
* Text Encryption & Decryption
* Secure Data Processing

## Objectives

This project aims to:

* Demonstrate the inner workings of cryptographic algorithms.
* Provide educational implementations with clear code structure.
* Help developers understand encryption, decryption, hashing, and authentication techniques.
* Serve as a reference for learning cryptography and secure programming in C++.

## Technologies Used

* C++17 / C++20
* Standard Template Library (STL)
* Object-Oriented Programming (OOP)
* Modular Programming
* Cryptographic Mathematics

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/FutureTechSecurity/Cryptography.git
cd cryptography
```

### Compile

```bash
g++ main.cpp -o cryptography
./cryptography
```
